# GitHub Copilot 介绍

**GitHub Copilot** 是由 GitHub 和 OpenAI 共同开发的人工智能代码辅助工具，可以自动地生成高质量代码片段和提供上下文信息。通过自然语言处理和机器学习技术，它能够根据程序员编写的代码、注释和上下文信息，自动生成代码，减轻程序员的工作量，节省开发者的时间和精力。安装后，在开发程序时可以大幅提高开发效率。
# GitHub Copilot 插件安装

##在 IntelliJ IDEA 中，选择 `Settings -> Plugins`，搜索插件 "**GitHub Copilot**"，直接安装即可，安装完毕后重启。
![2006.png](https://www.freeimg.cn/i/2024/01/16/65a6a66e905b7.png)

## VS Code 下 GitHub Copilot 插件安装
选择插件扩展，搜索 "GitHub Copilot"，安装并重启 IDE，需要登录 GitHub 账户进行身份验证。
[![2007.png](https://www.freeimg.cn/i/2024/01/16/65a6a66e8aede.png)

# 激活插件

## Windows 激活

在 Windows 系统双击执行 `cocopilot.bat`。
![2008.png](https://www.freeimg.cn/i/2024/01/16/65a6a66d12244.png)

若看到 "done. please restart your ide." 表示激活成功。

> **温馨提示：**
> - VSCode，Windows 系统执行 `vscode.bat`，
> - VSCode 中插件更新后需要重新执行脚本

## VS Code 安装和激活

*安装插件
破解工具压缩包里面有VSCode的插件安装包，如下
![2009.png](https://www.freeimg.cn/i/2024/01/16/65a6a66d089a6.png) 

直接导入插件包，然后重启VSCode就可以了

然后选择菜单 View -> Command Palette... -> 打开命令面板，输入「Enable CoCopilot」，点击确定，并重新加载...
![0038.png](https://www.freeimg.cn/i/2024/01/16/65a6a78fe5782.png)
